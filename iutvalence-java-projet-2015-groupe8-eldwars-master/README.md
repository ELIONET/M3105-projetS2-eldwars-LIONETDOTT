# iutvalence-java-projet-2015-groupe8-eldwars
Projet Java du groupe 8 - EldWars

Eldwars est un jeu de stratégie au tour par tour. Il s'agit de deux joueurs s'affrontant sur un ordinateur, chacun ayant pour but d'anéantir l'équipe adverse.

Pour lancer le jeu, il faut éxecuter le Main dans le package fr.iutvalence.groupe8.eldwars, entrer son nom et il est ensuite possible de jouer.

Déroulement d'une partie : 

-Les deux joueurs ont des commandants dans les coins opposés diagonalement lors du début de la partie.
-Chaque joueur recrute des unités en fonction de son or disponible
-Phase stratégique pour éléminer les unités adverses
-La partie se termine lorsque un joueur n'a plus d'unités.

Fonctionnalités :

-Attaquer une unité
-Recruter des unités
-Déplacer des unités
-Passer un tour.
